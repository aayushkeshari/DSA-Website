Sorting

Sorting means arranging data in order (like smallest to largest). Sorting is useful for searching faster, organizing results, and making data easier to read.

Bubble Sort (basic idea)

Bubble sort compares neighboring items and swaps them if they are out of order. It repeats this until the list is sorted. It is simple, but not efficient for large datasets.

Typical time complexity: O(n^2)

Merge Sort (basic idea)

Merge sort splits the list into smaller parts, sorts each part, then merges them back together. It uses a divide-and-conquer approach and is much faster than quadratic sorting for large inputs.

Typical time complexity: O(n log n)

Other Sorting Algorithms

  • Insertion Sort: Builds the sorted list gradually
  • Quick Sort: Uses divide-and-conquer with a pivot
  • Heap Sort: Uses a binary heap structure
